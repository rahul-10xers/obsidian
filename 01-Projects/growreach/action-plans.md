# Stage-Based Action Plans & Playbooks

This file contains ready-to-execute action plans organized by growth stage, budget level, and time availability. Every action is founder-validated with attribution.

## MASTER STAGED PLAYBOOK

### STAGE 1: PRE-LAUNCH (Product doesn't exist yet)

**Week 1-2: Validate Before Building**
1. Spend 45 minutes on Reddit mining exact customer language using advanced search: "[niche] + frustrated by" [Steph France — validated in 45 min]
2. Create a one-page landing page with benefit-first headline (NOT "future of productivity") [Margot — 2.5x conversion difference]
3. Add a $5 "Queue-Jump" prepayment to validate real buyer intent [Giovanni Brees — $45K MRR]
4. OR sell 50 lifetime licenses via email list before building [Gil — $20K in 3 days]
5. OR conduct "fake sales calls" asking if they'd pay your target price [Alex Becker — confirmed $3K annual willingness]

**Week 3-4: Build MVP**
1. Use Cursor/Lovable to vibe-code a functional MVP in 48 hours to 2 weeks [Hassam — 48hrs; Brees — weekend; Nick — 7 days]
2. Include ONLY the core "aha moment" output (the one thing that delivers value)
3. Exclude: billing systems, onboarding flows, analytics dashboards, complex settings [Becker, Brees, Nick all validated]
4. For AI tools: Consider BYOK (Bring Your Own Keys) to eliminate server costs [Prrenit — $80K revenue]

**Validation Decision Tree:**
- If pre-sale sells out → BUILD IT (Gil's path)
- If $5 deposits come in → BUILD IT (Brees's path)
- If fake sales calls get "Yes" → BUILD IT (Becker's path)
- If none of the above → PIVOT the idea or ICP (don't build)

### STAGE 3: 10 → 100 USERS

**Weekly Rhythm (20 hrs/week):**

Weekend: Product (4 hrs)
- Review screen recordings for UX friction [Nick]

### STAGE 4: 100 → 1,000 USERS

**Layer in These Channels:**

YouTube Creator Partnerships [Loic]:
1. Search niche keywords on YouTube; find creators with 10%+ engagement and 100+ comments
2. Look for creators who worked with a brand 3+ times (signals profitable partnerships)
3. Send 7-email outreach sequence + Instagram/Twitter DMs
4. Offer "Test Deal": one video at 30% discount from standard rate
5. If test is profitable → sign multi-video package
6. Consider equity co-founder model for top creators

Affiliate/Referral Program [Robinson, Dustin]:
1. Launch 20% lifetime commission affiliate program
2. Target agencies already using your product for their clients
3. Incentivize beyond cash: recognition, networking, custom perks
4. One person can manage the entire UGC/affiliate program [Robinson]

Waitlist & Scarcity Launches [Hoffman, Costa]:
1. Build email waitlist to 5K-10K via content
2. Launch capped beta (500 spots) with discounted LTD
3. Use webinars to close people "on the fence"
4. Increase price for next cohort → creates FOMO

Bottom-of-Funnel SEO [Hoffman, Aayush]:
1. Create "Alternatives to [Competitor]" pages
2. Create "[Competitor A] vs [Competitor B]" comparison pages
3. Target KD < 20, SV > 500
4. This is a 6-12 month play but compounds → autopilot traffic

## REDDIT MARKETING 4-WEEK LAUNCH PLAN

**Week 1: Warming & Infrastructure**
- Create personal account (not brand). Join 15 niche subreddits
- Comment 2-3x/day. Goal: 100 karma
- Set up F5bot/Gummy Search for keyword alerts
- Use one account per browser to avoid footprinting [Roman]

**Week 2: Value-First Engagement**
- Answer "Solution Request" threads with long-form help (no links)
- Create waitlist-only landing page for DM/comment referrals
- Identify top 3 most responsive subreddits by engagement

**Week 3: Soft-Sale Stories**
- Post first "Story" or "Guide" — focus on failure or learning experiment
- Mention SaaS only as tool used, or put link in comments only [Roman, Diego]
- Launch viral giveaway on X to capture emails [Gil]

**Week 4: Launch & Conversion**
- Drop short video demo in most responsive subreddit [Aayush]
- Saturday morning for dev niches, Tuesday for automation [Paul]
- Reply to every comment to stay at top of feed
- Run LTD launch for first 500 waitlist users [Hoffman]
