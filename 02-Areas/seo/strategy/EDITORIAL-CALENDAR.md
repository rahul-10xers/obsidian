## NEURONWRITER WORKFLOW (per piece)

```
1. Take keyword from TOPICAL-MAP.md
2. Run: node scripts/neuronwriter-research.js --keyword "target keyword"
3. Note NW score of top competitors
4. Write content using NW-required NLP terms
5. Check score — target 60+ before publishing
6. Push to WordPress as draft via REST API
7. Visual QA → publish
8. IndexNow submission (Rank Math auto-handles)
```
